위의 그림에서 instance 라고 되어진 부분이 바로 메모리 부분

#### 오라클 메모리 
- SGA - system global area
	- 오라클 데이터 베이스가 서버에서 실행되어 꺼질 때까지 유지됨
	- 여러사용자가 동시에 같은 오라클 데이터 베이스에 접혹하여 sGA 안의 데이터를 공유하여 shared global area 라고도함
