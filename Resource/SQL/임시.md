limit 키워드를 이용한 방법

```sql
select f.flavor
from (select flavor, sum(total_order) as ts
     from first_half
     group by flavor) f,
     (select flavor, sum(total_order) as ts
     from july
     group by flavor) j
where f.flavor = j.flavor
order by (f.ts + j.ts) desc
limit 3;

```
